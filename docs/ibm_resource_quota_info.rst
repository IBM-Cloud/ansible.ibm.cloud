
ibm_resource_quota_info -- Manage ibm_resource_quota info.
==========================================================

.. contents::
   :local:
   :depth: 1


Synopsis
--------

This module retrieves one or more ibm_resource_quota(s).



Requirements
------------
The below requirements are needed on the host that executes this module.

- ResourceManagerV2



Parameters
----------

  id (optional, str, None)
    The id of the quota.
